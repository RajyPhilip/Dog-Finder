
# Dog Finder App

 A simple Dog finder  app built with MERN stack. It allows users to mark their favourite dog , check the results of marked dogs .

## Features

- Mark favourites 
- View all results
- login/signup
-  Logout
## Technologies

- MongoDB
- Express.js
- React
- Node.js
